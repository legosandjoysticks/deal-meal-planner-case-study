# Security And Source Boundary

This public repo is a portfolio case study only.

## Included

- Product narrative
- Architecture overview
- Screenshots
- Demo GIF
- Safe API examples
- Resume and interview framing

## Excluded

- Source code
- Ingestion automation
- Store-specific scraping/click logic
- Generated databases
- Raw flyer PDFs or exports
- Logs
- `.env` files
- API keys
- Admin credentials
- Cloudflare credentials
- Tokens, cookies, or private operational notes

## Why

The goal is to show product engineering judgment without publishing implementation details that may later become part of a private product, paid tool, or selective open-source release.
